A pipeline, not a sales funnel.

Every LJ site is a single self-contained HTML file: hand-tuned typography, real photos when we have them, an embedded map, and a call button. We build it speculatively. You only pay if you keep it.

1DISCOVER

Find the business.

We sweep the LA area for small businesses with no website — or only a Facebook page. Real places that deserve a real front door.

$ run.py 1 # discover one
2DESIGN

Build the mockup.

Photos, hours, menu, testimonies — all real. One archetype per site, hand-picked to fit the place. No templates.

$ generate # single index.html
3SHIP

Publish to Pages.

The mockup goes live on a clean URL within minutes. We hand it to the owner. They can see it, click it, call themselves from it.

$ run.py --publish
4HAND OVER

If they want it, it's theirs.

Flat fee, no monthly. We transfer the repo, point the domain, and stay on call for the first month. Otherwise, it just stays online as our portfolio.

$ git remote set-url

about

We're a two-person studio in Pasadena. We started LJ Sites because we kept walking into restaurants and shops we loved that didn't show up on Google — and we got tired of that.

So we built a pipeline: discover places without a real online front door, ship them a single-page sales-pitch site, and let the work speak for itself. No cold outreach, no proposals, no contracts to sign before you've seen what we'd actually make for you.

Every site is hand-tuned. We don't ship templates. We don't take retainers. If you keep it, we transfer the repo and disappear.
